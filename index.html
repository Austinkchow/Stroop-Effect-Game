Color Game

The famous "Stroop Effect" is named after J. Ridley Stroop who discovered this strange phenomenon in the 1930s. Here is your job: click the color tiles of the following words. Do NOT read the color of the words...rather, read the words. For example, if the word "blue" is printed in a red color, you should click "blue". Click the colors as fast as you can. It is not as easy as you might think!

Round 1

Select the color tile that correlate with the word. If you get the right answer, you get 1 point. Otherwise, you lose 1 point.

Round 3

Select the color tile that correlate with the word. If you get the right answer, you get 2 points. Otherwise, you lose 2 points.

Round 2

Select the color tile that correlate with the word. If you get the right answer, you get 3 point. Otherwise, you lose 3 point.
